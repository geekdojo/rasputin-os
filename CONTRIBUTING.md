# Contributing

Thanks for your interest in Rasputin! The project is **pre-alpha** and
moving fast — issues, questions, and discussion are very welcome, and so is
patience with churn.

## Issues first

For anything non-trivial, please open an issue before starting a pull
request so we don't collide with in-flight work. Bug reports with
reproduction steps are gold.

## Pull requests

- Fork, branch, and open a PR against `main`.
- Keep PRs focused — one change per PR.
- Match the existing code style of whatever you're touching.

## Contributor License Agreement

Contributions require signing the [geekdojo CLA](CLA.md) — you keep your
copyright; the CLA grants geekdojo the licenses it needs (see the document
for the exact terms).

You don't need to do anything up front: your **first pull request triggers
the CLA bot**, which comments with instructions. Signing is a single reply
comment, and **one signature covers all geekdojo repositories**. Project
maintainers and the owner are exempt (their contributions are covered
already).
